## How to Run

```bash

```

### For React Projects:

**1. Clone the repository:**
```bash
git clone https://github.com/imanmaulana1/scrimba-solo-projects
```

**2. Navigate to the project folder:**
```bash
cd project-folder-name
```

**3. Install the dependencies:**
```bash
npm install
```

**4. Run the project:**
```bash
npm run dev
```
